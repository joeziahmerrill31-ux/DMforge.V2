  DM FORGE  –  D&D Session Prep Tool
  Pure HTML / JavaScript  –  No Python, No Server Required
════════════════════════════════════════════════════════════

YOUR 3 FILES
────────────────────────────────────────────────────────────
  index.html   ← The whole app (all pages)
  style.css    ← All styling + 6 colour themes
  app.js       ← All logic, monster data, dice, factions

That's it. No Flask, no pip, no server needed.


HOW TO OPEN LOCALLY (just to test)
────────────────────────────────────────────────────────────
Put all 3 files in the same folder, then double-click
index.html. It opens in your browser and works immediately.


HOW TO PUT IT LIVE ON THE INTERNET (FREE)
────────────────────────────────────────────────────────────
We use GitHub Pages — completely free, no sleeping, no limits.
Your URL will be:  https://yourusername.github.io/dmforge

STEP 1 – Make a free GitHub account
  → Go to  github.com  and sign up

STEP 2 – Create a new repository
  → Click the green "New" button
  → Name it:  dmforge
  → Make it Public
  → Click "Create repository"

STEP 3 – Upload your 3 files
  → On the repository page, click "uploading an existing file"
  → Drag and drop all 3 files: index.html, style.css, app.js
  → Click "Commit changes"

STEP 4 – Enable GitHub Pages
  → Click "Settings" tab (top of repository)
  → Click "Pages" in the left sidebar
  → Under "Branch" select:  main  and  / (root)
  → Click Save

STEP 5 – Wait ~60 seconds then visit:
  → https://yourusername.github.io/dmforge

🎉 Your site is live! Share the link with your players.


UPDATING THE SITE AFTER MAKING CHANGES
────────────────────────────────────────────────────────────
1. Go to your repository on github.com
2. Click the file you changed (e.g. app.js)
3. Click the pencil (edit) icon
4. Paste your new code
5. Click "Commit changes"
6. Wait 30 seconds and refresh your site


WANT A CUSTOM .COM DOMAIN?
────────────────────────────────────────────────────────────
GitHub Pages supports custom domains for FREE (you just
pay for the domain name itself, ~$12/year on Namecheap).

1. Buy your domain at  namecheap.com
2. In GitHub Settings → Pages → Custom Domain, type it in
3. In Namecheap DNS, add a CNAME record:
      Host:  www
      Value: yourusername.github.io
4. Wait a few hours for DNS to update
